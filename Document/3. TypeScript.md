# 类型声明

## 简单类型

### boolean

```ts
let bool: boolean = true
bool = false
```

### number

```ts
let num: number = 5
num = 0b1010
```

### string

```ts
let str: string = "hello"
str = "world"
```

### any

```ts
let foo: any = "abc"
foo = 1
foo = true
foo = undefined
```

### void

如果函数没有返回值，那么返回值的类型就是 void

```ts
function fun(): void {
  // ...
}

const fun = (): void => {
  // ...
}
```

## 复杂类型

### 数组类型

在元素类型后面加上 "[]"

```ts
const arr: number[] = [1, 2, 3]
```

使用数组泛型

```ts
const arr: Array<number> = [1, 2, 3]
```
