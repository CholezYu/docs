## 算法概述

### 时间复杂度

比较次数与数组长度 n 的关系。

$O(1)$ < $O(log_2n)$ < $O(n)$ < $O(nlog_2n)$ < $O(n^2)$ < $O(n^3)$ < $O(2^n)$ < $O(n!)$ < $O(n^n)$

### 空间复杂度

消耗内存与数组长度 n 的关系。

### 稳定性

如果 A 和 B 的值相等，但排序后 A、B 的次序保持不变，则这种算法是稳定的。

### 冒泡排序

```java
int[] arr = { 34, 54, 3, 2, 84, 65, 7, 19, 5, 76, 67 };

for (int i = 0; i < arr.length - 1; i++) {
  boolean flag = true;
  
  for (int j = 1; j < arr.length - i; j++) {
    if (arr[j - 1] > arr[j]) {
      int temp = arr[j - 1];
      arr[j - 1] = arr[j];
      arr[j] = temp;
      
      flag = false;
    }
  }
  
  if (flag) {
    break;
  }
}
```
